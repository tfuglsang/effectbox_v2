## **2.1 Codification**

The meaning of the codification is explained in *[Table](#page-0-3) 2*.

**Table 2. Codification explanation** 

<span id="page-0-3"></span>

| STM32TTXXY-DISCZ | Description                                                                                                   | Example: STM32H747I-DISCO     |
|------------------|---------------------------------------------------------------------------------------------------------------|-------------------------------|
| STM32TT          | MCU series in STM32 32-bit Arm Cortex MCUs                                                                    | STM32H7 series                |
| XX               | MCU product line in the series                                                                                | STM32H747/757 product line    |
| Y                | STM32 flash memory size:<br>– I for 2 Mbytes                                                                  | 2 Mbytes                      |
| DISCZ            | Toolkit configuration:<br>– DISCO: Discovery kit with LCD module<br>– DISC1: Discovery kit without LCD module | Discovery kit with LCD module |

<span id="page-0-0"></span><sup>2.</sup> Main board.

<span id="page-0-1"></span><sup>3.</sup> STMod+ fan-out expansion board.
